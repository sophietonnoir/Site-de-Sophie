Key To Key

Front End / Back End

Previous Next


Le développement du site s'est fait dans le cadre de l'apprentissage par projet à l'ISEP. Dans le cadre du développement du site nous avons développé diverses compétences comme le développement en HTML5, CSS3, PHP, SQL, Javascript ainsi que le cycle en V, la modélisation UML, et l'utilisation de Github.
Le site permet à un client d'échanger gratuitement son logement avec un autre utilisateur en France ou à l'étranger.
Un système de messagerie entre propriétaires et un forum sont disponibles sur le site.

  • Durée : Octobre 2014 - Janvier 2015
  • Client: Matthieu Manceny

Jeu de tanks

Le développement du jeu s'est fait dans le cadre de l'apprentissage du Java à l'ISEP.
Dans le cadre du développement du logiciel/jeu, nous avons appris: le développement en Java, les diagrammes de classes et d'utilisation en UML.
Le jeu permet à un ou plusieurs joueurs de s'affronter à travers l'utilisation de blindés répartis sur un terrain. Chaque joueur peut choisir soit de se déplacer, soit de tirer sur son adversaire.

  • Durée : Mars - Juin 2015
  • Client: Sylvain Lefebvre
